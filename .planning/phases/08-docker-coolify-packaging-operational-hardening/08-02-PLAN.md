## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Coolify UI → container env | Secrets injected at runtime via compose ${VAR}; must not enter build layers |
| orchestrator stop signal → worker | SIGTERM must reach the node process (init:true + exec-form) within the grace window |
| two services → one /data volume | Only web migrates; both read/write the shared WAL'd DB + uploads |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-04 | Denial of Service | SIGTERM swallowed by npx/sh PID 1 → SIGKILL mid-row | mitigate | exec-form `node worker.js` + `init: true`; raised stop_grace_period gives the drain time |
| T-08-05 | Information Disclosure | secret placed in a build arg | mitigate | secrets kept in `environment` (runtime) only; documented in .env.example |
| T-08-06 | Tampering | migration race between web + worker on start | mitigate | only web migrates (entrypoint); `depends_on: [web]` on worker |
| T-08-SC | Tampering | npm/pip/cargo installs | accept | No package changes in this plan |
</threat_model>
